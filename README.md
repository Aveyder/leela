# Leela
Leela is an open-source MMO game

## Get Started
### Prerequisites
- Node `v14.x`

### Installing
```console
npm install
```

### Running
To run live development builds, use:
```console
npm run dev
```

To run production builds, use:
```console
npm run build
```

Then, to run each production build, use:
```console
npm start
```
